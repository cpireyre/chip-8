<h1>CHIP-8 interpreter in C89</h1>
<p>
I thought this sounded like a fun project, and it is.
Work in progress.
</p>
<h2>How to use</h2>
`make`, then `./chip8 <rom.ch8>`.
Find ROMs online, see references.
